Turns your photoblog, into a online boutique, a fully enabled e-commerce site.

== Description ==

Fotomoto allows you to quickly and easily turn your photoblog into a fully featured e-commerce store. Sign up at [http://www.fotomoto.com](http://www.fotomoto.com) to receive your unique key and use it to activate the plugin and your pictures and photos will automatically have a buy me button under each of them. 
This plugin adds the required javascript for enabling the Fotomoto store on all pages.

== Installation ==

1. Download the plugin file.
2. Get your site key from the Fotomoto Dashboard found under Site > Settings
3. Unzip it to the /wp-content/plugins/ directory
4. Activate the plugin through the "Plugins" menu in Wordpress
5. Then under the settings menu select Fotomoto and enter the site key.
